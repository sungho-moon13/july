# 게임 개발 계획서

## 1. 게임 주제

- 게임 제목: 폰 저장공간 구조대
- 핵심 주제: 핸드폰 저장 용량을 쉽게 이해하고 정리하는 게임
- 한 줄 소개: 사진, 영상, 앱 캐시 같은 파일을 정리해서 핸드폰 용량을 확보하는 게임.
- 이 게임을 만들고 싶은 이유: 핸드폰 용량이 왜 부족해지는지 알고, 어떤 파일을 보관·백업·삭제해야 하는지 쉽게 배우고 싶다.

## 2. 학습 목표

- 플레이어가 게임을 통해 알게 될 내용:
- 꼭 설명하고 싶은 저장 용량 개념:
  - KB: 킬로바이트. 작은 문서 파일이나 아주 작은 이미지의 크기를 나타낼 때 쓰는 단위.
  - MB: 메가바이트. 사진, 음악 파일, 짧은 영상, 앱의 크기를 나타낼 때 자주 쓰는 단위.
  - GB: 기가바이트. 긴 영상, 게임 앱, 스마트폰 저장 공간을 나타낼 때 많이 쓰는 단위.
  - 클라우드 백업: 큰 파일을 인터넷 저장공간에 옮겨 핸드폰 용량을 확보하는 방법.
  - 앱 캐시: 앱이 임시로 저장한 데이터로, 지워도 되는 경우가 많아 용량 정리에 도움이 되는 파일.
- 저장 용량을 비유할 대상: 핸드폰 안의 저장공간
- 게임을 끝낸 뒤 플레이어가 할 수 있어야 하는 것: 사진, 영상, 앱 캐시, 문서가 저장공간을 차지한다는 것을 이해하고 알맞게 정리하는 것.

## 3. 대상 플레이어

- 주요 대상 연령: 초등학교 고학년 ~ 중학교 1학년
- 컴퓨터 지식 수준: 컴츄터 지식이 없어도 가능
- 플레이 시간이 얼마나 걸리면 좋을지: 플레이어가 버틸 수 있는 만큼 계속 진행
- 플레이어가 어려워할 수 있는 부분: 알맞은 창고에 파일 아이템을 넣는 것

## 4. 게임 장르와 방식

- 게임 장르: 핸드폰 저장공간 정리 게임
- 조작 방식: 마우스로 파일 아이템을 끌어서 보관, 클라우드 백업, 삭제 상자에 넣기
- 화면 구성: 왼쪽에는 뒤섞인 사진·영상·앱 캐시·문서 아이템, 오른쪽에는 보관/백업/삭제 상자를 배치
- 플레이 흐름: 시간이 지날수록 새 파일이 계속 들어오고, 플레이어는 저장공간이 꽉 차기 전에 빠르게 정리하기

## 5. 게임 세계관과 이야기

- 배경 설정: 핸드폰 저장공간이 사진, 영상, 앱 캐시, 다운로드 파일로 거의 가득 찬 상황
- 주인공: 핸드폰 용량을 지키는 초보 저장공간 관리자
- 플레이어의 역할: 파일의 종류와 용량을 보고 보관할지, 백업할지, 삭제할지 결정하는 역할
- 저장 용량을 설명하기 위한 이야기 설정: 큰 영상과 앱 캐시는 저장공간을 많이 차지하고, 중요한 파일은 함부로 지우면 안 됨
- 게임 시작 상황: 핸드폰 용량 막대가 거의 가득 차서 새 사진을 찍기 어려운 상태
- 게임 종료 상황: 저장공간이 꽉 차면 게임이 끝나고, 플레이어가 얼마나 오래 버텼는지와 점수를 확인함

## 6. 핵심 게임 규칙

- 플레이어가 해야 하는 일: 사진, 영상, 앱 캐시, 문서 아이템을 알맞은 보관/백업/삭제 상자에 정리하기
- 저장 용량 개념이 규칙에 들어가는 방법: 용량이 큰 영상은 백업하면 저장공간이 줄어들고, 필요 없는 캐시나 중복 사진은 삭제하면 저장공간이 확보됨
- 점수 계산 방식: 파일마다 10점씩, 연속 성공하면 콤보 보너스 추가
- 제한 시간 또는 제한 횟수: 제한 시간은 없고, 핸드폰 저장공간이 꽉 차면 종료
- 난이도가 올라가는 방식: 시간이 지날수록 파일 아이템이 더 빠르게 들어오고, 한 번에 들어오는 아이템 수도 늘어난다.
- 추가 규칙: 저장공간 사용량이 0GB가 되면 새 파일이 조금 더 빠르게 들어와서 다시 용량이 차오르게 된다.

## 7. 게임 진행 구성

### 7.1 기본 진행

- 배울 개념: 저장공간, 파일 용량, 보관/백업/삭제
- 목표: 핸드폰 용량을 차지하는 파일을 알맞게 정리하는 방법을 이해하기.
- 라운드 이름: 실시간 폰 저장공간 비우기
- 제한 시간: 없음
- 등장하는 파일 또는 아이템:
  - 보관 아이템: 가족 사진, 숙제 파일, 연락처 백업
  - 백업 아이템: 여행 영상, 긴 동영상, 음악 앨범
  - 삭제 아이템: 중복 사진, 앱 캐시, 오래된 다운로드, 지난 캡처
- 배치 방식: 파일 아이템들이 화면 왼쪽에 실시간으로 들어오고 뒤엉켜서 쌓인다.
- 정리할 창고: 보관 상자, 클라우드 백업 상자, 삭제 상자
- 성공 조건: 최대한 오래 버티면서 파일을 많이 맞게 정리하고 핸드폰 사용 용량을 줄이기.
- 재미 요소: 연속으로 맞히면 콤보 점수가 올라가고, 백업/삭제에 성공하면 저장공간 막대가 줄어든다.

### 7.2 난이도 상승 방식

- 초반: 파일이 약 1~2초마다 1개씩 들어온다.
- 중반: 시간이 충분히 지난 뒤 파일이 조금 더 빠르게 들어오고, 가끔 2개씩 동시에 들어온다.
- 후반: 오래 버틴 플레이어에게만 사진, 영상, 캐시, 다운로드 파일이 여러 개씩 들어온다.
- 종료 조건: 저장공간 막대가 100%가 되면 게임 종료.
- 보정 규칙: 저장공간이 0GB까지 비워지면 파일 유입 속도가 잠깐 빨라져 게임 흐름이 계속 이어진다.

### 7.3 추가 아이디어

- 아이디어 1: 저장공간이 90% 이상 차면 경고 효과음과 붉은 화면 표시
- 아이디어 2: 일정 시간마다 아주 큰 영상 파일이 들어오는 이벤트
- 아이디어 3: 도움 아이템으로 "자동 캐시 삭제" 버튼 제공
